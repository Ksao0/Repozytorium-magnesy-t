1.0.0.0.2.26 Wstępna
Zaleca się aktualizowanie programu przed każdym użyciem
Wydano: 20.04.2023

Jest to wersja wstępna naszego programu liczącego. Aktualizacje będą wypuszczane częściej lub wcześniej niż w wersji stabilnej, ale mogą występować drobne błędy.
W każdej chwili możesz wrócić do stabilniejszej wersji beta.
Po wybraniu tej wersji zainstaluj wymagane biblioteki:
 - os
 - tkinter
 - messagebox
 - tkinter
 - scrolledtext
 - datetime
 - urllib
 - request
 - subprocess
 - requests
Aby to zrobić otwórz terminal cmd i wpisz:
 pip install (nazwa biblioteki)
 
Na przykład:
 pip install tkinter

Zmiany względem poprzedniej wersji wstępneej:
 - Poprawki błędów
 - Możliwość usunięcia informacji o poprzednich obliczeniach
 - Możliwość zmiany kosztów przyjętych przez program
