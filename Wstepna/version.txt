Zmiany względem poprzedniej wersji wstępnej:

   - pygame
 - Dodano opcję tworzenia wykresu na podstawie danych z poprzednich obliczeń
 - Program automatycznie zgłasza wykryte błędy (ta opcja jest dostępna tylko dla wybranych użytkowników)
 - Program obsługuje niektóre problemy związane z internetem
 - Możliwość używania wszystkich okien jednocześnie
 - Możliwość zgłaszania błędów z poziomu programu (ta opcja jest dostępna tylko dla wybranych użytkowników)
Aby to zrobić otwórz terminal cmd i wpisz:
 pip install (nazwa biblioteki)
 
Przykład:
 pip install tkinter
