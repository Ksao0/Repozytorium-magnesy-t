1.0.0.1.7 Wstępna
Status: Poprawka wersji
Wydano: 03.05.2023

Jest to wersja wstępna naszego programu liczącego. Aktualizacje będą wypuszczane częściej niż w wersji stabilnej, ale mogą występować drobne błędy.
W każdej chwili możesz wrócić do stabilniejszej wersji beta.
Po wybraniu tej wersji zainstaluj wymagane biblioteki:
 - os
 - tkinter
 - messagebox
 - scrolledtext
 - datetime
 - urllib
 - request  (dwie biblioteki o podobnej nazwie)
 - requests (dwie biblioteki o podobnej nazwie)
 - subprocess

Aby to zrobić otwórz terminal cmd i wpisz:
 pip install (nazwa biblioteki)
 
Na przykład:
 pip install tkinter

Zmiany względem poprzedniej wersji wstępneej:
 - Poprawki wersji są pobierane automatycznie
