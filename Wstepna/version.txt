Zmiany względem poprzedniej wersji wstępnej:

 + WAŻNE Dnia 13.05.2023 dodano następujące biblioteki. Pobierz je poleceniem pip install nazwa biblioteki:
   - sys
   - traceback
   - github
   - time
Aby to zrobić otwórz terminal cmd i wpisz:
 pip install (nazwa biblioteki)

 pip install tkinter
